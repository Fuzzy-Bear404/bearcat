# bearcat
bearcat is a Django Web-app

Django template made with Python, JavaScript, HTML/CSS, Tailwind CSS, and MySQL.

This is a Django template that is free to edit and use for your own projects and website

macOS/Linux:

source venv/bin/activate

venv\Scripts\activate

Create a .env file with the following example content (replace values as necessary):

SECRET_KEY=

DB_NAME=
DB_USER=
DB_PASSWORD=
DB_HOST=
DB_PORT=

Migrate the Database

python manage.py migrate

Collect Static Files

python manage.py collectstatic

Running the Django Development Server

python manage.py runserver
